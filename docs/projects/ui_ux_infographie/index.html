Créer une infographie UI vs UX (180 points)

Objectif​

Créer une infographie claire et visuelle qui explique la différence entre UI (User Interface) et UX (User Experience) Design.

Recherche et compréhension​

Avant de commencer, assurez-vous de bien comprendre les concepts :

UI (User Interface Design)

  • Se concentre sur l'apparence visuelle
  • Éléments : couleurs, typographie, boutons, icônes
  • Objectif : rendre l'interface esthétique et cohérente

UX (User Experience Design)

  • Se concentre sur l'expérience globale de l'utilisateur
  • Éléments : recherche utilisateur, architecture de l'information, parcours utilisateur
  • Objectif : rendre le produit intuitif et facile à utiliser

Questions à explorer :

  • Quelles sont les 3 principales différences ?
  • Comment ces deux disciplines collaborent-elles ?
  • Quels outils utilisent respectivement les designers UI et UX ?

Planification du contenu​

Décidez quelles informations inclure dans votre infographie :

Sections suggérées :

  1. Titre accrocheur
  2. Définitions courtes de UI et UX
  3. Comparaison visuelle (tableau ou diagramme)
  4. Métaphore illustrative (ex : UI = décoration d'une maison, UX = plan architectural)
  5. Outils populaires pour chaque discipline
  6. Citation ou fait intéressant

Conseil : Limitez le texte. Une infographie efficace est 70% visuelle, 30% texte.

Création de la structure​

Choisissez votre outil :

  • Canva (gratuit, facile)
  • Figma (professionnel)
  • Adobe Illustrator (avancé)
  • PowerPoint ou Google Slides (accessible)

Structure recommandée :

  • Format vertical (800 x 2000 pixels minimum)
  • Divisez l'espace en 5-6 sections horizontales
  • Utilisez une grille pour aligner les éléments

Palette de couleurs :

  • Choisissez 2-3 couleurs principales
  • Suggestion : une couleur pour UI, une pour UX, une neutre pour le texte

Design visuel​

En-tête​

  • Titre principal : "UI vs UX : Quelle est la différence ?"
  • Sous-titre optionnel
  • Icône ou illustration d'introduction

Corps principal​

Créez une section comparative avec :

Colonne UI :

  • Icône représentative (palette, écran, pinceau)
  • Définition en 1-2 phrases
  • 3-4 mots-clés (ex : visuel, interface, esthétique, branding)
  • Exemple d'outils (Sketch, Figma, Adobe XD)

Colonne UX :

  • Icône représentative (personne, parcours, carte)
  • Définition en 1-2 phrases
  • 3-4 mots-clés (ex : recherche, ergonomie, parcours, wireframes)
  • Exemple d'outils (Miro, Maze, UserTesting)

Section métaphore​

Illustrez avec une analogie visuelle :

  • Restaurant : UI = la décoration et le menu, UX = la qualité du service et l'expérience
  • Voiture : UI = le design extérieur, UX = le confort de conduite
  • Créez votre propre métaphore originale !

Collaboration​

Montrez comment UI et UX travaillent ensemble :

  • Diagramme de Venn
  • Ou flèche cyclique montrant le processus itératif

Checklist de vérification​

Compréhension UI (User Interface)​

  • Définition claire de l'UI Design
  • Focus sur l'apparence visuelle démontré
  • Éléments UI mentionnés : couleurs, typographie, boutons, icônes
  • Objectif explicité : esthétique et cohérence

Compréhension UX (User Experience)​

  • Définition claire de l'UX Design
  • Focus sur l'expérience globale démontré
  • Éléments UX mentionnés : recherche utilisateur, architecture d'information, parcours
  • Objectif explicité : intuitivité et facilité d'utilisation

Questions explorées​

  • Identification des 3 principales différences UI/UX
  • Explication de la collaboration entre les deux disciplines
  • Liste des outils spécifiques à chaque discipline

Sections incluses​

  • Titre accrocheur et impactant
  • Définitions courtes de UI et UX (1-2 phrases max)
  • Comparaison visuelle (tableau/diagramme)
  • Métaphore illustrative pertinente
  • Outils populaires pour chaque discipline
  • Citation ou fait intéressant (optionnel)

Équilibre contenu​

  • Proportion respectée : ~70% visuel, ~30% texte
  • Texte limité et concis
  • Hiérarchie de l'information claire
  • Contenu pertinent et non redondant

Choix de l'outil​

  • Outil de création sélectionné (Canva/Figma/Illustrator/Slides)
  • Maîtrise suffisante de l'outil choisi

Format et dimensions​

  • Format vertical utilisé
  • Dimensions minimum : 800 x 2000 pixels (ou équivalent)
  • Résolution adaptée (72 dpi web minimum)

Organisation spatiale​

  • Espace divisé en 5-6 sections horizontales
  • Utilisation d'une grille pour alignement
  • Marges et espacements cohérents
  • Flux de lecture logique (haut → bas)

Palette de couleurs​

  • 2-3 couleurs principales choisies
  • Une couleur distincte pour UI
  • Une couleur distincte pour UX
  • Couleur neutre pour le texte
  • Cohérence chromatique sur toute l'infographie
  • Contraste suffisant pour la lisibilité

En-tête​

  • Titre principal présent et visible
  • Titre explicite (ex: "UI vs UX : Quelle est la différence ?")
  • Sous-titre si pertinent
  • Icône ou illustration d'introduction
  • Hiérarchie visuelle claire

Éléments visuels​

  • Icône représentative de l'UI (palette/écran/pinceau)
  • Design cohérent avec le thème UI

Contenu textuel​

  • Définition en 1-2 phrases maximum
  • 3-4 mots-clés pertinents inclus
  • Exemples : visuel, interface, esthétique, branding

Outils​

  • Liste d'outils UI (ex: Sketch, Figma, Adobe XD)
  • 2-4 outils mentionnés
  • Outils actuels et pertinents

Éléments visuels​

  • Icône représentative de l'UX (personne/parcours/carte)
  • Design cohérent avec le thème UX

Contenu textuel​

  • Définition en 1-2 phrases maximum
  • 3-4 mots-clés pertinents inclus
  • Exemples : recherche, ergonomie, parcours, wireframes

Outils​

  • Liste d'outils UX (ex: Miro, Maze, UserTesting)
  • 2-4 outils mentionnés
  • Outils actuels et pertinents

Comparaison visuelle​

  • Format clair (tableau/colonnes/diagramme)
  • Différences visuellement distinguables
  • Équilibre entre les deux côtés
  • Lecture facile et rapide

Métaphore choisie​

  • Analogie visuelle présente
  • Métaphore claire et compréhensible
  • Illustration de la métaphore
  • Légende explicative si nécessaire

Exemples de métaphores​

  • Restaurant (UI = déco/menu, UX = service/expérience)
  • Voiture (UI = design extérieur, UX = confort)
  • OU métaphore originale créée

Qualité de la métaphore​

  • Pertinence pour le public cible
  • Simplicité de compréhension
  • Renforce la compréhension des concepts

Visualisation de la collaboration​

  • Représentation visuelle de l'interaction UI/UX
  • Diagramme de Venn OU
  • Flèche cyclique/processus itératif OU
  • Autre représentation pertinente

Clarté du message​

  • Montre l'interdépendance UI/UX
  • Indique un processus de travail
  • Évite l'opposition stricte

Typographie​

  • Maximum 2-3 polices différentes
  • Hiérarchie typographique claire (titres/sous-titres/texte)
  • Tailles de police lisibles
  • Cohérence dans toute l'infographie

Iconographie​

  • Style d'icônes cohérent
  • Icônes claires et reconnaissables
  • Taille appropriée
  • Alignement avec la palette de couleurs

Composition​

  • Équilibre visuel
  • Espaces blancs utilisés efficacement
  • Pas de surcharge visuelle
  • Guidage du regard optimisé

Accessibilité​

  • Contraste texte/fond suffisant
  • Texte lisible (taille minimum 10-12pt)
  • Couleurs distinguables
  • Compréhensible sans couleur (pour daltoniens)

Export et format​

  • Format d'export approprié (PNG, PDF, JPG)
  • Résolution finale vérifiée
  • Taille de fichier raisonnable
  • Qualité visuelle préservée

Vérification finale​

  • Aucune faute d'orthographe
  • Informations exactes et à jour
  • Tous les éléments alignés
  • Cohérence visuelle totale
  • Message principal clair et mémorable
