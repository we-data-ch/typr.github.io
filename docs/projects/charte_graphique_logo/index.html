Magasin de vente de fruits (180 points)

Logo fourni : Une orange stylisée avec des feuilles vertes, entourée d'un double cercle orange sur fond sombre.

PARTIE 1 : ANALYSE DU LOGO​

Étape 1 : Observation et description​

Analysez attentivement le logo et répondez aux questions suivantes :

  1. Quels sont les éléments visuels présents ?

    • Listez tous les composants du logo

  2. Quel est le style graphique ?

    • Réaliste / Semi-réaliste / Minimaliste / Illustratif / Autre ?

  3. Quelles émotions/valeurs ce logo véhicule-t-il ?

    • Fraîcheur, modernité, nature, qualité, etc.

  4. Quelle est la cible probable ?

    • Famille, jeunes urbains, consommateurs bio, etc.

PARTIE 2 : PALETTE DE COULEURS​

Étape 2 : Extraction des couleurs principales​

À faire : Identifiez et notez les couleurs du logo avec leurs codes hexadécimaux approximatifs.

Créez votre palette :

Proposez 2-3 couleurs supplémentaires qui s'harmonisent avec la palette pour enrichir la charte :

PARTIE 3 : TYPOGRAPHIE​

Étape 4 : Choix des typographies​

Le logo suggère un style moderne et frais. Choisissez :

Police pour les titres (Display) :

  • Nom de la police : __
  • Justification : __
  • Exemple d'utilisation : Nom du magasin, titres de sections

Police pour le corps de texte :

  • Nom de la police : __
  • Justification : __
  • Exemple d'utilisation : Descriptions, prix, informations

Consignes :

  • Taille minimum : ___pt
  • Interlignage recommandé : ___
  • Graisse disponibles : Regular, Bold, etc.

PARTIE 4 : DÉCLINAISONS DU LOGO​

Étape 5 : Créez différentes versions​

Dessinez ou décrivez les variations suivantes :

  1. Version principale (déjà fournie)
  2. Version monochrome (noir et blanc uniquement)
  3. Version simplifiée (pour petits formats : favicon, app mobile)
  4. Version horizontale (avec nom du magasin à côté)

PARTIE 5 : ÉLÉMENTS GRAPHIQUES​

Étape 6 : Créez des éléments de support​

Inspirez-vous du logo pour créer :

  1. Motifs/patterns :

    • Proposez un motif répétitif utilisant des formes du logo (cercles, feuilles, etc.)

  2. Icônes :

    • Dessinez 5 icônes dans le même style pour : livraison, bio, local, fraîcheur, promotion

  3. Formes géométriques :

    • Définissez des formes récurrentes (cercles, courbes organiques, etc.)

PARTIE 6 : CHARTE FINALE​

Étape 7 : Document de synthèse​

Compilez tous les éléments dans un document comprenant :

  1. Page de couverture avec le logo
  2. Introduction : identité de la marque, valeurs, ton
  3. Palette de couleurs avec codes et usages
  4. Typographie avec exemples
  5. Déclinaisons du logo avec règles d'utilisation
  6. Espacements et marges autour du logo
  7. Exemples d'applications correctes et incorrectes
  8. Éléments graphiques complémentaires

Checklist : Création de la charte graphique du magasin de fruits​

Étape 1 : Observation et description​

  • Lister tous les éléments visuels du logo (orange, feuilles, cercles, etc.)
  • Identifier le style graphique (réaliste, minimaliste, etc.)
  • Déterminer les émotions et valeurs véhiculées
  • Définir la cible probable du magasin

Étape 2 : Extraction des couleurs principales​

  • Identifier la couleur primaire avec son code hexadécimal
  • Identifier les ombres/contrastes avec leur code hexadécimal
  • Identifier la couleur secondaire avec son code hexadécimal
  • Identifier les accents avec leur code hexadécimal
  • Identifier la couleur de fond/texte avec son code hexadécimal
  • Compléter le tableau avec tous les codes hex et leurs rôles

Étape 3 : Couleurs complémentaires​

  • Proposer une première couleur supplémentaire harmonieuse
  • Proposer une deuxième couleur supplémentaire harmonieuse
  • Proposer une troisième couleur supplémentaire harmonieuse
  • Noter les codes hexadécimaux de ces couleurs

Étape 4 : Choix des typographies​

Police pour les titres (Display)​

  • Choisir une police adaptée au style du logo
  • Justifier le choix de cette police
  • Définir les exemples d'utilisation

Police pour le corps de texte​

  • Choisir une police lisible et cohérente
  • Justifier le choix de cette police
  • Définir les exemples d'utilisation

Consignes typographiques​

  • Définir la taille minimum en points
  • Définir l'interlignage recommandé
  • Lister les graisses disponibles (Regular, Bold, etc.)

Étape 5 : Créer les variations​

  • Version principale (vérifier qu'elle est bien documentée)
  • Créer/décrire la version monochrome (noir et blanc)
  • Créer/décrire la version simplifiée (favicon, app mobile)
  • Créer/décrire la version horizontale (avec nom du magasin)

Étape 6 : Créer les éléments de support​

Motifs/patterns​

  • Proposer un motif répétitif basé sur les formes du logo
  • S'assurer qu'il est cohérent avec l'identité visuelle

Icônes (5 au total)​

  • Créer/dessiner l'icône "livraison"
  • Créer/dessiner l'icône "bio"
  • Créer/dessiner l'icône "local"
  • Créer/dessiner l'icône "fraîcheur"
  • Créer/dessiner l'icône "promotion"
  • Vérifier que toutes les icônes ont le même style graphique

Formes géométriques​

  • Définir les formes récurrentes (cercles, courbes, etc.)
  • Documenter leur utilisation

Étape 7 : Document de synthèse​

Structure du document​

  • Page 1 : Couverture avec le logo
  • Page 2 : Introduction (identité, valeurs, ton de la marque)
  • Page 3 : Palette de couleurs complète avec codes et usages
  • Page 4 : Typographie avec exemples concrets
  • Page 5 : Déclinaisons du logo avec règles d'utilisation
  • Page 6 : Espacements et marges autour du logo
  • Page 7 : Exemples d'applications correctes
  • Page 8 : Exemples d'applications incorrectes (à éviter)
  • Page 9 : Éléments graphiques complémentaires

Vérification finale​

  • Toutes les pages sont complètes
  • Les exemples visuels sont clairs
  • Le document est cohérent et professionnel
  • Les règles d'utilisation sont explicites
